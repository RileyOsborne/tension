# Tension

A trivia game built with Laravel and Livewire.

## Requirements

- Docker

## Getting Started

```bash
docker compose up -d --build
```

The app will be available at http://localhost:8888

Migrations run automatically on first start. An APP_KEY is generated if not provided.

## Stopping

```bash
docker compose down
```

## Tech Stack

- Laravel 12
- Livewire 3 / Volt
- Vite
- SQLite
